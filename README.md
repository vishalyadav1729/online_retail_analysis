# Online Retail Customer Analysis

This project analyzes customer purchase behavior using the UCI Online Retail dataset.

## Tools Used
- Python
- Jupyter Notebook
- pandas
- matplotlib
- scikit-learn

## Project Goals
- Clean a real retail dataset
- Perform exploratory data analysis
- Build RFM customer features
- Segment customers using KMeans

## Files
- `notebook.ipynb` — full analysis
- `outputs/customer_segments.csv` — final customer segments

## Dataset
UCI Online Retail Dataset
